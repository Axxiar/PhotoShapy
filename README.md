# PhotoShape 
PhotoShape est un éditeur basique d'images en python


## Actions :
- [ ] Charger des images (en cours, presque finit)
- [ ] Sauvegarder sous différents formats
- [ ] Ajuster les couleurs RGB
- [ ] Dessiner
- [ ] Redimensionner
- [ ] Mode Clair/Sombre
- [ ] Raccourcis Clavier

Ajouter des Filtres :

- [ ] Effet de Flou
- [ ] Effet noir & blanc
- [ ] Effet ASCII
- [ ] Effet Pastel (comme si l'image avait été peinte)
